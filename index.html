URL指定の使い方
このページは、URLの末尾にパラメータを付けると初期値を指定できます。
使うパラメータは下の6つ。 ?w=作業用動画ID&b=休憩用動画ID&wm=作業分&bm=休憩分&ws=作業開始秒&bs=休憩開始秒 例：
w = 作業動画ID / b = 休憩動画ID / wm = 作業時間（分） / bm = 休憩時間（分） / ws = 作業ループ開始秒 / bs = 休憩ループ開始秒
YouTubeの動画IDは、たとえば https://youtu.be/jfKfPfyJRdk の jfKfPfyJRdk の部分。
